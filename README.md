# Isometric-NCE
The goal of this project is to create a lightweight 2.5D game that simulates Nepal College of Engineering and helps new students understand the campus layout.

## Objective

- Game Type: Isometric open-world
- Art Style: Low-poly Style
- Game Map: Grid-based movement

## Features

- Enteral building
- NPCs (AI)
- Interactive Activities

## Development Process

- Start simple
- Build layer by layer and chunk by chunk
- Focus on gameplay first, graphics and polish and optimization come latern
